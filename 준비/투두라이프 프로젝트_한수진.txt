프론트엔드 비동기 API 프로젝트

1. 프로젝트명 : 1인자의 날씨요정

2. 핵심가치(서비스 내용)
   1) 빨래, 청소할때 팁을 나타내기 
     (가능하다면, 클릭시 전체 내용 볼 수 있게 팁 백과사전 느낌으로 만들기)

   2) 날씨
      : 오늘 포함+2~3일 날씨(이미지,기온, 습도)

   3) 디데이 -> (가능하다면,  반대로 청소한지 며칠이나 지났는 지도 확인할 수 있도록 설정) 
      : 주기적으로 해야하는 집안일(분리수거, 화장실청소 등) 입력 가능하게 작성하기

   4) 투두할일
      : 오늘의 날씨(비소식) -> 우산챙기기, 외출시 환기를 위한 창문 개폐 유무 추천(이때 가능하다면, 미세먼지 농도 API도 이용해보기)
      : 오늘 포함+2~3일 날씨(기온, 습도) -> 오늘 빨래를 하기 적합한 날씨인지 추천 지수 표현, 제습기/가습기 가동 추천 

      -> 추천 사항중 강력 추천 사항은 투두에 기록(이때 사용자가 직접 입력하는 목록이랑은 색상 구분)!
    
3. 작업기간
   7/8 ~ 7/14
